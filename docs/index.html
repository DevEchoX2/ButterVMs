ButterVMS Static Site

Browser VMs for your servers

ButterVMS gives your users browser-based desktops with simple tier controls. Free sessions run for 45 minutes, paid sessions run for 8 hours.

Free Tier

45-minute browser VM sessions for onboarding and lightweight usage.

Paid Tier

8-hour browser VM sessions for full workdays and premium users.

Features

  • ✓ Browser-first VNC desktop access
  • ✓ Free tier: 45-minute sessions
  • ✓ Premium tier: 8-hour sessions
  • ✓ Automatic session expiry & cleanup
  • ✓ BTC payment reference workflow
  • ✓ Per-user isolated VM containers

Deploy your own

ButterVMS is fully open-source. Deploy to your servers with Docker Compose.

docker compose up -d --build
